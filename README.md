# Attendance & Shrinkage Tracker

This project demonstrates a performance tracking system designed to monitor attendance, shrinkage, and coaching effectiveness in a team environment.

## Key Features
- Automated shrinkage calculation
- Daily and monthly performance tracking
- Coaching and feedback management
- Dashboard for performance insights

## Impact
- Reduced manual effort in reporting
- Improved accuracy in tracking attendance
- Helped identify performance gaps and improve team productivity

(Note: Dummy data is used for demonstration purposes)
